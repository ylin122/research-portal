The user has a standard research brief prompt they run for every company. It covers:
- Section 0: Recent updates (market-moving news only)
- Section 1: Company overview (brief)
- Section 2: Key business / products (lead with how they make money)
- Section 3: Customer focus (profile, verticals, geography, buying drivers)
- Section 4: Industry & market (headwinds/tailwinds, TAM, growth estimates)
- Section 5: Competitive landscape (competitors, advantages, threats/opportunities)
- Section 6: Recent transactions (reverse chronological, material only)
- Section 7: Financials — intentionally left empty

Tone: Direct, honest, concise. Bullet format, no periods. No marketing language. Written as if briefing a portfolio manager. The user is in investment/PE research.
